CRM-IT · Registro de trabajo

Fecha: 2026-03-14

Tareas realizadas:
- Se inicializó un proyecto nuevo con Next.js, TypeScript y ESLint.
- Se aisló un repositorio Git local dentro de esta carpeta para no mezclar este proyecto con el home del usuario.
- Se definió un stack base robusto y simple de mantener: Next.js + SQLite local para la primera etapa.
- Se creó el esquema inicial de datos para planes, clientes y facturación mensual de soporte con `better-sqlite3`.
- Se cargaron los planes TASS Starter, Pro y Elite con sus valores de licencia y soporte mensual.
- Se creó una semilla inicial con clientes demo para probar flujos reales.
- Se construyó un dashboard inicial con branding de IT-Integral Solutions.
- Se implementó un alta de cliente con creación automática del primer ciclo de soporte.
- Se implementó una acción para marcar pagos mensuales de soporte y generar el siguiente vencimiento.
- Se modularizó la interfaz en páginas separadas: dashboard, clientes, soporte y productos.
- Se agregó una navegación lateral persistente para que el CRM se lea como sistema y no como una única pantalla.
- Se separaron componentes reutilizables para métricas, productos, clientes, soporte y alta comercial.
- Se agregó un módulo de pagos separado con métricas de cobranza y vista financiera dedicada.
- Se implementaron acciones para registrar cobro del 50% inicial y del 50% final de cada cliente.
- Se validó nuevamente la aplicación con `npm run lint` y `npm run build`.
- Se generalizó el CRM para que TASS sea tratado como producto y no como el sistema completo.
- Se agregaron productos base orientados por rubro para viajes, restaurantes y soluciones corporativas.
- Se creó `requirements.txt` con prerrequisitos de entorno para levantar el CRM en otra PC.
